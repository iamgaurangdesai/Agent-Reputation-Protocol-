## Project Overview

**Agent Reputation Protocol (ARP)** is the first on-chain reputation system for AI agents. It solves the fundamental problem of trust in the agent economy.

### The Problem
AI agents need to transact with strangers. Without reputation, every interaction is risky. How do you know an agent will deliver?

### The Solution
ARP provides a unified trust score combining:
- **On-chain reputation** (transaction history, ratings)
- **USDC staking** (economic commitment)
- **Ethos credibility** (human verification)
